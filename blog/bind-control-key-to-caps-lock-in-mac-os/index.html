Bind the Control key to the Caps Lock in macOS

Do you ever use the Caps Lock key? I bet you ignore this nice button and don’t use it at all, unless you write on COBOL or FORTRAN.

This button is located on one of the most used keyboard lines and your little finger is almost always above this place. But you never press it. Fortunately we can change the action of this button and you are able to bind the action that you often use in your daily routine.

As for me I’m very pleased with the use the Control key action on the Caps Lock key. I’m an active user of vim and tmux and this binding allows me to be more efficient in my work.

Of course, you can use Karabiner to rebind this key but I recommend you to look in the System Prefernces first. Go to System Preferences -> Keyboard -> Keyboard -> Modifier Keys and there choose the Control key action for the Caps Lock key. You can even choose the Esc key, that could be useful if you have a MacBook Pro with Touch Bar.
